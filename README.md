# react-redux-helloworld
Hello, World! app in React and Redux

## Run the app

```
git clone https://github.com/vaibhavmule/react-redux-helloworld.git

cd react-redux-helloworld
npm install
npm start


If you are using Yarn

cd react-redux-helloworld
yarn install
yarn start

open http://localhost:3000/

```
